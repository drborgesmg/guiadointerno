# 🩺 Guia Médico — Receitas & Evoluções

Ferramenta clínica de referência rápida para médicos, internos e estudantes de medicina. Desenvolvida para uso no dia a dia do pronto-socorro e consultório.

---

## 📋 O que tem no guia

### 💊 Receitas por Doença
Mais de **90 condições clínicas** organizadas por especialidade, com opções terapêuticas, doses, posologia e orientações ao paciente.

| Especialidade | Exemplos |
|---|---|
| Cardiologia | SCA, FA, TSV, IC/EAP, Sepse |
| Neurologia | Enxaqueca, Crise convulsiva, Meningite |
| Pneumologia | Asma, DPOC, Pneumonia, TB |
| Gastroenterologia | HDA, Pancreatite, Apendicite, DRGE |
| Nefrologia | ITU, Pielonefrite, CAD, Cólica renal |
| Infectologia | Dengue, Leptospirose, Sífilis, Sepse |
| Dermatologia | Celulite, Escabiose, Herpes, Micoses |
| Ortopedia | Lombalgia, Gota, Fratura exposta |
| Ginecologia | ISTs, Candidíase, DIP, Anticoncepção |
| Psiquiatria | Agitação, Crise de pânico, Psicose |
| Pediatria | Doses pediátricas, GECA, Bronquiolite |
| Emergência | Intubação (SRI), Intoxicações, Ofidismo |

### 🔍 Busca por Medicação
Digite o nome de qualquer medicamento e veja todas as doenças e doses em que ele aparece.

### 📋 Evoluções Clínicas
10 modelos prontos para adaptar:
- Exame físico padrão
- Consulta de rotina
- Urgência / Pronto-atendimento
- Alta hospitalar
- Retorno / Reinternação
- Dor aguda (analgesia escalonada)
- Pediátrico
- Ginecológico / Obstétrico
- Abdome agudo
- Sepse / Choque séptico

---

## ⚡ Funcionalidades

- **Offline** — funciona sem internet após carregar a página
- **Busca em tempo real** por doença ou medicação
- **Filtro por especialidade** com um clique
- **Copiar com 1 clique** — copia a receita formatada para a área de transferência
- **Responsivo** — funciona no celular e no desktop
- Arquivo HTML único, sem instalação, sem login

---

## 🚀 Como usar

1. Acesse o link acima pelo celular ou computador
2. Busque a doença ou o medicamento desejado
3. Clique no card para expandir e copiar a receita

---

## ⚠️ Aviso

Este guia é uma ferramenta de referência clínica para internos de saúde habilitados. Não substitui anamnese clínico, protocolos institucionais nem diretrizes atualizadas. Adapte sempre à realidade do seu paciente.
